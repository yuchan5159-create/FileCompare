#(C# 코딩) FileCompare


## 개요
-C# 프로그래밍 학습
-1줄 소개: 폴더 안의 파일 비교와 복사
-사용한 플랫폼:
 -C#, .NET Windows Forms, Visual Studio, GitHub
-사용한 컨트롤:
 -Label, button, SplitContariner, Panel
-사용한 기술과 구현한 기능
 -Visual Studio를 이용하여 UI디자인
 -
## 실행 화면
 -코드의 실행 스크린샷과 구현 내용 설명
